Python 序列化

序列化在程序运行过程中，所有变量都是在内存中，比如，定义一个dict: a = dict(name="Ann",age=21,dcore=98) 我们可以随时修改变量，比如把 score 改成 60 ,但是一旦程序结束，变量所占用的内存就被操作系统全部回收。如果没有把修改后的 60 存储到磁盘上，下次重新运行程序，变量又被初始化为 98 。 我们把变量从内存中变成可存储或传输的过程称之为序列化 ，在Python中叫pickling，在其他语言中也被称之为serialization，marshalling，flattening等等，都是一个意思。 序列化之后，就可以把序列化后的内容写入磁盘，或者通过网络传输到别的机器上。 反过来，把变量内容从序列化的对象重新读到内存里称之为反序列化，即unpickling。 Python提供了pickle模块来实现序列化。 将文件序列化并写入文件： >>> import pickle >>> a = dict(name='Ann', age=21, score=98) >&

slots的使用

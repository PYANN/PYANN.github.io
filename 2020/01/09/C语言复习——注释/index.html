C语言复习——注释

    # include <stdio.h>             // 编译预处理指令
    int main()                      // 定义主函数
    {                               // 函数开始标志
        printf("Hello C !\n");      // 在控制台输出指定的信息
        return 0;                   // 函数执行完毕时返回函数值0
    }                               // 函数结束标志

C语言两种注释方式：

(1) 以 // 开始的单行注释。如上面介绍的注释。这种注释可以独占一行，也可以出现在一行中其他内容的右侧。此种注释的范围从 // 开始，以换行符结束。也就是说这种注释不能跨行。如果注释内容一行内写不下，可以用多个单行注释，如果下面两行是连续的注释行：
    // 如果注释内容一行写不下
    // 可以在下一行重新用'//'，然后继续写注释。
(2) 以/*开始，以*/结束的块式注解。这种注解可以包含多行内容。它可以单独占一行（在行头以/*开始，行末以/*结束），也可以包含多行。编译系统在发现一个/*后，会开始找结束符*/，把二者间的内容作为注释。

注意：//和/**/在字符串中，都不作为注释，而是作为字符串的一部分。
    如：
        printf("//hello world");
    或者
        printf("/*hello world*/");
    输出分别是：
        // hello world
    和
        /*hello world*/
